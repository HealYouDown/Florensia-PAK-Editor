# Florensia-PAK-Editor
.pak editor for Florensia.

## Usage
You can either use the menu and open a file from the editor or register it as the default tool to open .pak files.

Rightclicking an item will open a context menu where you can either extract this specific file or all files from this .pak.

To add a file, just drag it in the left list. If the filename already exists, it will be overwritten, otherwise added.

## Releases
You can download the compiled version [here](https://github.com/HealYouDown/Florensia-PAK-Editor/releases).
